# Job Finder App React

A modern Job Finder web application built with React that allows to browse jobs, apply filters. Save jobs that are stored in localstorage in another page called bookmarks. This app also allows to delete the saved jobs. The app fetches real API for the jobs. The Job Finder web app also has a profile that shows personal informations about the user that is logged in. This app allows to navigate using react router dom and context api for the profile and informations. The app is fully responsive and works on any device. Professional, scalable structure and modern design that follows real-world development practices. 

# Features

- Responsive design

- easy navigation

- multiple pages

- fast and clean code

- reusable code

- filter jobs by: company, location, salary, job-title

- Bookmark favorite jobs 

- React Routing
